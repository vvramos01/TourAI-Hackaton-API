# TourAI
Roteirização Inteligente de Pontos Turísticos
TourAI é uma API desenvolvida pela equipe AltTech durante um hackathon em dezembro de 2025. O projeto tem como objetivo criar roteiros turísticos personalizados de forma inteligente, considerando preferências do usuário, localização e condições simuladas em tempo real.

# O que o projeto faz?
Gera roteiros personalizados com base nas preferências do usuário (cultura, natureza, gastronomia etc.)

Ordena os pontos turísticos por proximidade, otimizando o deslocamento

Simula condições como lotação e clima para tornar as recomendações mais realistas

Oferece uma interface web para interação com o usuário

# Tecnologias utilizadas
Backend: Java com Spring Boot (API REST)

Frontend: JavaScript, CSS e HTML

Banco de Dados: H2 em memória (para testes)

Ferramentas: Maven, Git

# Estrutura
O repositório contém tanto o backend (API) quanto o frontend (/tourai-frontend), formando uma solução completa para demonstração do conceito.
